# Xenoguesser

**An engine that peers into hyperspace and discovers alien lifeforms from hidden causal worlds.**

Xenoguesser does not hand you a dossier. It hands you *instruments*. Every alien
is generated by a hidden **Worldseed** — planetary and stellar first principles —
that is then run, conceptually, through deep time:

```
celestial + planetary conditions → ecology → body plan → senses → cognition
   → social structure → culture → technology → self-modification → final form
```

You only ever see the **final form**, through six alien instruments. From it, you
reverse-engineer the causal chain back to the world that produced it. The final
form cannot lie: every trait is downstream of an earlier pressure.

> Each alien is a theorem with skin.

## Play

Open `index.html` in any modern browser. No build step, no dependencies.
(Or serve the folder statically, e.g. `python3 -m http.server`.)

- **Read the instruments.** Morphology, metabolism, behavior, culture residue,
  singularity artifacts, and the planetary shadow each reveal a slice of the
  specimen — the shadow only ever gives *indirect* hints.
- **Reconstruct the causes.** Nine guess panels: planet type, star system, the
  dominant environmental pressures, the evolutionary pathway, cognitive
  adaptations, cultural consequences, whether they reached a singularity, how
  culture reshaped their biology, and the one constraint the final form can
  never escape.
- **Submit.** You get a score, a rank, and then the full hidden Worldseed and
  causal chain are revealed.
- **Share a seed** to hand someone the exact same specimen.

## How it works

The engine is grounded in real concepts — natural selection, convergent
evolution, niche construction, sensory ecology, metabolism and ecological
tradeoffs, symbiosis, eusociality, self-domestication, gene-culture coevolution,
catastrophe selection, and cultural feedback loops. It is **speculative
evolutionary inference, not biology protocol**: there are no engineering recipes,
lab methods, or actionable instructions anywhere in it.

Aliens are built by **constraint logic, not randomness**:

1. A hidden Worldseed is sampled from granular first principles — number of suns
   (one/two/three), moons (none/one/giant/many/rings), rotation, axial tilt,
   orbit shape, magnetic field, plus planet type, primary star, gravity,
   atmosphere, chemistry, climate stability, catastrophe regime, predator
   pressure, resources, terrain, biosphere age, and biosphere logic.
2. A **reconciliation pass** removes physical contradictions (a high-gravity
   super-Earth is never "low gravity"; a short-lived blue star never hosts an
   ancient biosphere; a tidally locked world is rotationally locked).
3. Each choice injects weighted **pressure tags** into a pool.
4. The pipeline runs stage by stage. A trait only fires if the accumulated
   pressures support it, and each fired trait emits new tags that feed later
   stages — so cognition is downstream of senses, which are downstream of the
   light regime, and so on.
5. Intelligence, technology, and singularity are **rungs with real attrition**.
   Roughly half of biospheres develop culture, ~1 in 5 reach technology, ~1 in 8
   cross a self-modification singularity — gated by a causal "encephalization"
   score, not a coin flip. So "did they reach a singularity?" is a genuine
   question.

### Scoring rewards causal reasoning, not trivia

Everything is scored on **concept-tag overlap**, never exact labels. Identify the
right *pressure* and you score well even if you pick the wrong category name — a
guess of "dark, unstable, radiation-soaked world with synchronized behavior"
scores highly against a true "tidally locked flaring red dwarf," because the
underlying pressures match.

Determinism: the same Worldseed string always produces the same alien.

## Project layout

| File | Role |
|------|------|
| `js/rng.js` | Seeded deterministic PRNG |
| `js/concepts.js` | The pressure/trait tag vocabulary (the causal currency) |
| `js/worldseed.js` | Planetary Worldseed dimensions |
| `js/celestial.js` | Celestial first-principles layer + reconciliation |
| `js/rules.js` | Causal trait rules for every pipeline stage |
| `js/engine.js` | Worldseed generation + the simulation pipeline |
| `js/scoring.js` | Guess construction + tag-overlap scoring |
| `js/instruments.js` | The six instruments + procedural creature glyph |
| `js/app.js` | Game flow, DOM rendering, the post-verdict reveal |

All speculative and non-actionable by design.
